# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be welcoming and inclusive.
- Be respectful of differing viewpoints and experiences.
- Accept constructive criticism gracefully.
- Focus on what is best for the project and the community.
- Show empathy toward other community members.

## Reporting

Unacceptable behavior can be reported by opening a private issue or emailing the maintainer listed in the repository. Reports are confidential.

Maintainers have the right to remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with this Code of Conduct.
